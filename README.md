# 🚀 Social Media & Chat Platform

This project is a full-stack Social Media and Chat application that allows users to connect, share posts, interact with others, and communicate in real time. It combines social networking features with messaging functionality to provide a complete user experience.

---

##  Features

-  User Authentication (Login/Register)
-  User Profiles
-  Create and manage posts
-  Like and comment on posts
-  Follow / Unfollow users
-  Real-time chat system
-  Messaging between users
-  Search users and chats
-  Responsive UI

##  Installation & Setup

### 1️ Clone the Repository
git clone https://github.com/your-username/social-media-platform.git
cd Social-Media-Platform-main

---

### 2️ Setup Frontend
cd chatapp-main
npm install
npm start

---

### 3️ Setup Backend
cd chatappmain_Backend/backend
npm install
npm start

---

### 4️ Database Setup
- Run SQL script: scripts/sql/001_init.sql
- Configure DB connection in backend files
